Keyboard:
W: avatar moving forward
S: avatar moving backward
A: avatar turning left
D: avatar turning right
V: toggle rendering xyz axes

Playstation  Gamepad:
Axis.RX --> yaw
Axis.RY --> pitch
Axis.Z --> zoom
